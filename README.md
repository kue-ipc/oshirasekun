# お知らせ君

京都教育大学専用のお知らせ君です。
[IPCニュース](https://www.kyokyo-u.ac.jp/c_ipc/data/note.txt)を
表示するだけです。

## 開発

- Rust
- Node.js + pnpm
- Tauri + Solid.js + Civet

```sh
pnpm install
pnpm tauri dev
```

## ビルド

```sh
pnpm install
pnpm tauri build
```
